# Daemon

A curated list of resources for a more literate programming inspired by the awesome collections of resources on Github.

## Philosophy

As programmers we face the challenge of the ever growing pool of knowledge. It seems as though as soon as you learn something, something else is starting to grow and you have to learn it too.

This is my list of resources so I can try to keep up with my demand for learning new things.

## Contributing

Contributing is pretty easy. If you think it's a resource that could possibly improve your knowledge or help you to better your skills, or if it's something that could improve the project, just add a pull request.

The only rule is: keep it organized.
